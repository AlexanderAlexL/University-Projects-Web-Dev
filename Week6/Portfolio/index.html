Services

Web Design

If you need a website but you don't know where to start, then you'll probably need UX and UI design to plan the Wireframes/prototypes, archtecture and also the visual design, colors and typography of your website. I can help you with this. I use Figma for the design process at this stage but I am familiar with other wireframing and prototyping design tools too. Web design is an important part for delivering the right message to your customers.

Web Development

Following the web design, I implement the design into code. For the front-end development I like using free-hand code such as vanilla HTML, CSS and JavaScript because of the unlimited opportunities with free-hand code. However, I am familiar with CSS frameworks and JS libraries such as Bootstrap, ReactJS (in progress). If your preference is a WordPress based website, I can help you with that too. Currently, I am focusing on the front-end development.

SEO

SEO or Search Engine Optimization is the process of making your website more competitive in the Search Engine Ranking Pages. One of the most importants search engines is Google. SEO is a complex process that requires keyword research, proper on-page SEO, off-page SEO and technical SEO. It's important to implement the appropriate principles during the design and development process too. Search Engine Optimization is a non-stop process.
